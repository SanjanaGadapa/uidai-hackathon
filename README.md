# Aadhaar Enrollment Analysis — Maharashtra

Analysis of Aadhaar enrollment, biometric update, and demographic 
activity data across 36 districts of Maharashtra.

## Tech Stack
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

## Key Analysis
- District-wise enrollment and biometric update trends
- Age group composition across districts
- Anomaly detection using z-score and 95th percentile threshold
- Monthly time-series trends
- Linear regression to predict biometric updates from enrollment volume
